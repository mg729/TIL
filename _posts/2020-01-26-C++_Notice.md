### C++ Notice
*  **malloc 과 new의 차이**  
   * malloc : 메모리 동적 할당
   * new : 메모리 동적할당 + 생성자 호출  
* **구조체와 클래스 차이**  
   * structure : 접근지정자가 default로 *public*  
   * class : 접근지정자가 default로 *private*  
* **C++ conctructor 호출**
  1. *stack*에 객체선언후 생성자 호출
  ```c++
  class Animal
  {
    ...
  };
  int main()
  {
    Animal a;
  }
  ```
  2. *heap*에 객체선언후 생성자 호출
  ```c++
  class Animal
  {
    ...
  };
  int main()
  {
    Animal a = new Animal;
  }
  ```
