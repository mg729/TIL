## git
```console
~$ git
```

- 위와같이 git을 리눅스 명령어 창에 입력했는데 git usage에 대한 안내문이 뜬다면 git프로그램을 다운받을 필요없음

```console
~$ sudo apt-get install git
```
- git 프로그램이 없는 경우 package manager를 활용하여 git을 다운

## git clone (외부 오픈소스 프로젝트 복제)
```console
~$ git clone https://github.com/~~.git [dir]
```
