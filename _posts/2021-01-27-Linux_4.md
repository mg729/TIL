## Package Manager
- package
  - program/application/app
  - 컴퓨터를 사용한다 == package의 기능을 사용한다.
  - 리눅스에서 기본적으로 컴퓨터에 포함되진않았지만 설치해야하는 경우 package manager를 제공
  - 모바일의 Appstore 역할
  - 리눅스의 대표적인 Package Manager
    - **apt**
    - **yum**

## apt (Advanced Package Tool)
```console
~$ sudo apt-get update;
```

- apt 최신상태로 업데이트하고 package 다운받기
